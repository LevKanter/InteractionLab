Welcome to Core: Interaction Lab! This class is first and foremost about making awesome websites. We will learn how to write HTML, CSS, and Javascript in order to produce sharp typography, multi-column layouts, navigation menus, image galleries, video players, overlays, and more! Rather than using WYSIWYG software like Dreamweaver, we will be writing and composing code ourselves. This is to achieve a more precise and fundamental understanding of how this stuff works.

If you have never written or seen HTML in your life, fear not. We will concentrate on the basics. Everything is going to be allright! If it happens that you already do have experience with these technologies, fear not, you won't be bored— we will hone your craft.

Since everyone in this class is also taking Core: Interaction Studio, emphasis will be placed on learning the technical things you'll need in order to execute your studio projects. At some points during the semester, we will be workshopping your core studio projects in class. The homework that I'll be assigning (generally on a weekly basis) is going to mainly consist of exercises and small projects that won't span more than two weeks.

There is no physical textbook for this class. Our textbook is going to be the entire Internet! We will be looking at websites not only for inspiration, but also to analyze and understand how they are structured.

Software Tools

The following applications and plug-ins are industrial strength and free. Note that there are many other FTP clients, text editors, etc, besides the ones listed here, and they all basically achieve the same stuff. If you prefer something that is not on this list, feel free to stick to that. The only exception is Firebug, which is required for this class.

  • Firefox

    Firefox will be our primary browser in this class because of its add-on, Firebug.

  • Firebug

    Add-on for Firefox that enables you to analyze and manipulate elements on a web page, directly in the browser. We will be using Firebug extensively both to understand how existing pages are structured, and to aid us in developing our own sites.

  • Cyberduck

    An FTP client. It enables you to upload files to, and download files from, a remote server.

  • TextWrangler

    Mac only

    A text editor that is well suited for writing HTML, CSS, and Javascript files (among others).

  • Komodo Edit

    Also a text editor, available on Windows, Mac, and Linux (whereas TextWrangler is only for Mac).
